A **visual evolution simulator** that demonstrates how organisms change over time through **genetic mutations and trait variations**.

The simulation creates a population of digital organisms whose characteristics evolve dynamically when mutations occur.

Built using **HTML5 Canvas and JavaScript**, this project visualizes the basic principles of **genetics and biological evolution**.

---

# 🌍 Project Overview

The **Genetic Evolution Simulator** represents a simplified biological ecosystem.

Each organism has traits such as:

• **Size**
• **Speed**
• **Color**

When mutations occur, these traits change slightly, creating **diversity within the population**.

Over time, the simulation demonstrates how populations evolve through **random genetic variation**.

---

# 🚀 Features

🧬 **DNA Helix Visualization**
A dynamic DNA-like structure appears in the background.

🌱 **Evolving Organisms**
Multiple organisms move across the environment.

🧪 **Mutation Trigger System**
Users can trigger mutations that change organism traits.

🎨 **Genetic Diversity Visualization**
Organisms evolve different sizes, speeds, and colors.

🎥 **Real-Time Canvas Animation**
Continuous simulation rendering using the Canvas API.

---

# 🧠 How the Simulation Works

The simulator models simplified evolutionary principles:

• Organisms are initialized with random traits
• Movement speed represents adaptability
• Mutations randomly modify traits
• Trait diversity appears over generations

This provides a **visual introduction to genetic variation and evolution**.
